# Projeto Site IPREMB

Este projeto é um site institucional do IPREMB em fase de front-end para apresentação e possível aprovação pela presidência.

## Objetivo atual

Finalizar o front-end institucional para demonstração, mantendo compatibilidade com GitHub Pages.

## Contexto do projeto

O site será apresentado ao presidente do IPREMB. Caso aprovado, poderá substituir o site atual. Nesta primeira etapa, o foco principal é layout, navegação, responsividade e apresentação institucional.

Posteriormente, será planejado o back-end e painel administrativo para gestão de conteúdos, como notícias, comunicados, documentos, fotos, relatórios, transparência e demais publicações.

## Regras obrigatórias

- Não reorganizar a estrutura de pastas nesta etapa.
- Não converter o projeto para framework.
- Não criar back-end nesta fase.
- Não remover páginas existentes.
- Não criar funcionalidades fora do escopo solicitado.
- Não alterar conteúdos institucionais além do necessário para layout, navegação e links.
- Manter compatibilidade com GitHub Pages.
- Preservar a identidade visual institucional do IPREMB.
- Priorizar boa apresentação em desktop e mobile.
- Antes de alterar arquivos, analisar o impacto e informar quais arquivos serão modificados.
- Executar alterações em etapas pequenas.
- Informar todos os arquivos alterados ao final de cada tarefa.
- Não inventar páginas, menus ou funcionalidades sem solicitação expressa.
- Não alterar links oficiais sem autorização.

## Diretriz de trabalho

Sempre que receber uma tarefa, primeiro analise os arquivos envolvidos, informe o plano de alteração e aguarde confirmação antes de modificar o projeto, salvo quando for expressamente autorizado a executar.

As alterações devem ser feitas com foco em estabilidade, apresentação institucional, clareza visual, responsividade e compatibilidade com GitHub Pages.
